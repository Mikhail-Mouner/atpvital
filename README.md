### Installation

```sh
git clone https://github.com/Mikhail-Mouner/atpvital.git
```

```sh
cd atpvital
```

```sh
cp .env.example .env
```

```sh
composer install
```

```sh
npm install && npm run dev
```

Create Table Name( atpvital )

```sh
php artisan migrate --seed
```

```sh
php artisan passport:install
```

```sh
pa schedule:work
```

```sh
php artisan serve
```
